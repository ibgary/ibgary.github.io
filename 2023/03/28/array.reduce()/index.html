array.reduce()方法

发表于2023-03-28|更新于2023-03-28|tech
|阅读量:

reduce() 是 JavaScript 数组的一个高阶函数，它可以用来对数组中的所有元素进行迭代并返回一个累加值。它接受一个回调函数作为参数，该函数将为每个元素执行一次，并返回一个累加值。该函数的形式如下：

其中，callback 是一个接受 4 个参数的函数，它将为数组中的每个元素执行一次，并返回一个累加值。参数如下：

  • accumulator：累加值，该值在每次调用 callback 时被更新。
  • currentValue：当前元素的值。
  • currentIndex：当前元素在数组中的索引。
  • array：原始数组对象。

第二个可选参数 initialValue 是作为第一次调用 callback 时的累加值。如果未提供初始值，则使用数组的第一个元素作为初始值，并跳过第一个元素的索引。如果数组为空且没有提供初始值，则会抛出 TypeError 异常。

需要注意的是，reduce() 是一个高阶函数，它可以使用一个回调函数来执行任何操作，例如计算最大或最小值、查找元素、连接数组等。
